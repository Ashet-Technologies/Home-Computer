# USB Expansion

The USB Expansion provides 4 USB 1.1 ports to the computer which are attached to the backplane.

## Pinout

| Pin | Name | Signal direction | Remark             |
|-----|------|------------------|--------------------|
| 1   | GND  | -                | Signal ground      |
| 2   | D+   | in/out           | Positive data lane |
| 3   | D-   | in/out           | Negative data lane |
| 4   | VBUS | -                | 5V                 |
